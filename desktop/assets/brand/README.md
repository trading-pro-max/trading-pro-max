# Trading Pro Max Desktop Brand Hand-off

This folder is reserved for final branded desktop release assets.

## Reserved Release Paths

- Product wordmark or lockup: `desktop/assets/brand/trading-pro-max-wordmark.svg`
- Windows application icon: `desktop/assets/brand/icons/trading-pro-max.ico`
- NSIS installer header art: `desktop/assets/brand/installer/trading-pro-max-installer-header.bmp`
- NSIS installer sidebar art: `desktop/assets/brand/installer/trading-pro-max-installer-sidebar.bmp`

## Usage Notes

- Keep the final `.ico` asset ready for both the executable and the Windows installer.
- Keep installer BMP assets aligned to the NSIS artwork slots so packaging can be finalized without touching product code.
- Do not replace these placeholders with marketing-site assets. This folder is for the desktop product release only.
